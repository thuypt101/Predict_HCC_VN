An-Intelligent-Machine-Learning-Approach-for-Predicting-and-Explaining-Hepatocellular-Carcinoma
This repository provides a complete workflow for analyzing and modeling the dataset, including feature selection, visualization, model training, evaluation, and explainability.
Files Overview
VD_HCC.xlsx — Raw dataset
Visualazation_VD_HCC.ipynb — General exploratory data visualization.
Feature_selection_VD_HCC_880.ipynb — Feature selection using statistical and ML-based methods.
ML_model_VD_HCC_880.ipynb — Experimenting with various machine learning models to identify a robust classifier and an optimal feature set
imbalance_handing_model_VD_HCC_880.ipynb — Assessing model robustness in imbalanced data scenarios
Compare_models_VD_HCC_880.ipynb — Visualizes and compares model performance on the test set.
graph_VD_HCC_880.ipynb — Graph-based visualization of feature importance
LIME_VD_HCC_880.ipynb — Generates local explanations and plots average feature importance from LIME.
SHAP_VD_HCC_880.ipynb — Visualizes feature values using SHAP.
